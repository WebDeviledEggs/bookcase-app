BookCase Server Activation Commands (PowerShell)

1. Activate Virtual Environment & Start Django Backend
# Navigate to backend directory
cd bookcase-app\backend

# Activate virtual environment
venv\Scripts\Activate.ps1
venv\Scripts\Activate

# If you get execution policy error, run this first:
# Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# You should see (venv) at the beginning of your prompt

# Start Django server
python manage.py runserver

# Django will run on http://localhost:8000

2. Start React Frontend (Open NEW PowerShell window)
# Navigate to frontend directory
cd bookcase-app\frontend

# Start React development server
npm start

# React will run on http://localhost:3000 and auto-open browser

Notes:
- Keep both PowerShell windows open while developing
- Django terminal shows API requests and backend errors
- React terminal shows frontend build info and hot reload status
- Use Ctrl+C to stop servers when done
